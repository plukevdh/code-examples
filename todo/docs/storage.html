  • ¶

    Storage.coffee is a simple wrapper and normalizer for storing objects in HTML5's localStorage. This could easily be extended to be a polyfill for localStorage and switch off to cookies if needed. For now, we only care about using localStorage.

    class Storage
  constructor: (@key="demo") ->
    @items = @_refresh()
